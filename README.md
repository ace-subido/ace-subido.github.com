# acesubido.com

## Development

```
$ brew install direnv
$ direnv allow

# Development Mode
$ jekyll s

# Release Mode
$ rake publish

# Write a draft
$ jekyll draft "<title>"

# Publish a draft
$ jekyll publish "_drafts/<title>"
```
